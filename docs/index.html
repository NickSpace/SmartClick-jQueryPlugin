SmartClick-jQueryPlugin Demo

1~3是绑定SmartClick的，4~6是绑定jQuery自带的click事件。
都是绑定单击事件的唯一的区别是选择文本的时候不同。
选择1，2，3行文本的时候不会触发单击事件
选择4，5，6行文本的时候会触发单击事件
